# JOR-V3.1 Visualization Tools

This directory contains the plotting engine for JOR Bayesian Fusion results. 

### 🚀 Usage Instructions
These scripts are designed to process the output from the main PyMC runner. For the scripts to find your data:

1. Ensure `jor_scores.csv` is present in the **root directory** of the project.
2. Run the scripts from the project root using:
   ```bash
   python viz/jor_forest.py
   python viz/jor_chronological.py
